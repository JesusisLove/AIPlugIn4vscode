# AI 助手插件安装手顺

---

## 首次安装

### 第一步：安装 VS Code（已安装可跳过）

前往 https://code.visualstudio.com 下载安装。

---

### 第二步：安装插件

1. 收到 `.vsix` 文件后，保存到任意位置（例如桌面）

2. 打开 VS Code

3. 点击左侧栏的「扩展」图标（四个方块的图标）

4. 点击右上角的「···」菜单

5. 选择「从 VSIX 安装...」

   ![从VSIX安装](从VSIX安装.png)

6. 选择收到的 `.vsix` 文件，点击「安装」

7. 安装完成后，VS Code 右下角会提示「已安装」

---

### 第三步：使用插件

- 快捷键 `Cmd + Shift + A`（Mac）打开 AI 助手
- 或按 `Cmd + Shift + P`，输入「打开 AI 助手」回车

---

## 更新插件（收到新版本时）

重新安装即可，步骤与首次安装完全相同。  
新版本会自动覆盖旧版本，**聊天记录不受影响**。
